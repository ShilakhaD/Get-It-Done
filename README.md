# Weekflow

A tile-based weekly roadmap for planning and tracking tasks across the week.

## Features

- Seven-day tile layout with per-day progress
- Add, edit, complete, and delete tasks
- Priority labels and optional notes
- Weekly progress summary
- Browser storage, so tasks persist between visits
- Responsive layout for desktop and mobile

## Run locally

Open `index.html` directly, or serve the directory:

```bash
python3 -m http.server 4173
```

Then visit [http://localhost:4173](http://localhost:4173).

## Deployment

This repository includes a GitHub Pages workflow. Every push to `main` publishes the
static site automatically.
